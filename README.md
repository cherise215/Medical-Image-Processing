## Install 
-Stable Version:

`pip install --upgrade pip`

`python -m pip install itk`

- Lastest Version

`pip install --upgrade pip`

`pip install itk -f https://github.com/InsightSoftwareConsortium/ITKPythonPackage/releases/tag/latest`
